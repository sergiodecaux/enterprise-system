# ENTERPRISE SYSTEM — Полный концепт и описание приложения

**Версия:** 2026-07-21  
**Бренд:** ENTERPRISE SYSTEM  
**Девиз:** «Математика против эмоций»  
**Формат:** Telegram Mini App / веб-терминал  
**Рынок данных:** MEXC USDT Perpetual (фьючерсы)

---

## 1. Суть продукта

**ENTERPRISE SYSTEM** — аналитический терминал для криптофьючерсов. Приложение **не торгует** и **не хранит API-ключи биржи**. Оно собирает рыночные данные, считает Smart Money / order-flow / news-контекст и сопровождает трейдера на всех этапах:

| Этап | Модуль | Что даёт |
|------|--------|----------|
| Отбор | **Sniper Mode** | Только элитные сетапы (score ≥ 7, R:R ≥ 1:3) |
| Мемы | **Meme Pulse** | Volatility Scalper для шиткоинов (топ-10) |
| Вход | **Confidence Score** | 4 фактора → 0–100% уверенности |
| Сопровождение | **Tactical Copilot** | Breakeven, стены, инвалидация, TP/SL |
| Обзор | **Radar** | Полный watchlist + контекст рынка |

Продукт позиционируется как **тактический терминал + копилот сделки**, а не как сигнальный канал и не как бот автоторговли.

## 2. Для кого

| Аудитория | Что получает |
|-----------|--------------|
| Интрадей / свинг на USDT-perp | Sniper-сетапы, SL/TP, сопровождение сделки |
| Трейдеры SMC / ICT | OB, FVG, OTE, MSS, CHoCH, PO3, Session DNA |
| Трейдеры order-flow | Стакан, стены, imbalance, Tape Momentum, Buyer Aggression |
| Мем-трейдеры | Meme Pulse: volume spike, gap, mean reversion, spread pressure |
| Контекстные трейдеры | Сессии, календарь NFP/FOMC, новостной sentiment |

Не целевая аудитория: инвесторы «купи и держи», алго-фонды (нет execution API), новички без базового понимания риска.

## 3. Принципы продукта

1. **Сигнал = confluence, не мнение.** Score 0–10 из зон и фильтров; вероятность — производная от score.
2. **Daily Bias сверху.** Если день «NO_TRADE» — система режет направление, а не «угадывает».
3. **Sniper ≠ Radar.** Radar показывает всё; Sniper — только прошедшие жёсткий фильтр.
4. **Мемы отдельно.** Volatility Scalper с другой логикой (скорость, агрессия, gap).
5. **Прозрачность.** DataLog, Confidence Score и зоны показывают *почему* score такой.
6. **Без ключей и без LLM в runtime.** Публичные источники + локальный keyword-sentiment.
7. **Мобильный first (TMA).** 4 вкладки + bottom drawer (Tactical).
8. **Сделки локально.** Active Trades в localStorage — сопровождение, не исполнение.

## 4. Что приложение делает / чего не делает

### Делает
- Сканирует watchlist MEXC (ядро 10 пар + поиск).
- Строит SMC-сетапы LONG/SHORT с SL, TP1, TP2.
- Фильтрует элитные сетапы в Sniper Mode.
- Сканирует мем-коины (вся вселенная MEXC → топ-10 в UI).
- Считает Confidence Score перед входом в сделку.
- Хранит активные сделки и мониторит их (Copilot).
- Показывает живой график, индикаторы, зоны, прогнозные пути.
- Читает стакан, ленту сделок, усиливает score.
- Накладывает торговые сессии и макро-календарь.
- Агрегирует новости, Fear & Greed, sentiment по монете.

### Не делает
- Не открывает и не закрывает позиции на бирже.
- Не подключается к кошельку / не custodian.
- Не обещает win-rate и не является финансовой рекомендацией.
- Не требует API-ключей MEXC от пользователя.
- Не запускает тяжёлые LLM в браузере.

## 5. Навигация и пользовательский путь

### 5.1 Четыре вкладки (App.tsx)

```
┌─────────────────────────────────────────────────────────────┐
│  Header — ENTERPRISE SYSTEM · статус связи                  │
│  NewsStrip — бегущая строка новостей (опционально)          │
├──────────┬──────────┬──────────┬──────────────────────────┤
│  СНАЙПЕР │   МЕМЫ   │  СДЕЛКИ  │         РАДАР            │
│ (default)│          │          │                          │
└──────────┴──────────┴──────────┴──────────────────────────┘
```

| Вкладка | Компонент | Назначение |
|---------|-----------|------------|
| **Снайпер** | `SniperView` | Элитные сетапы, Confidence Score, «Открыть сделку» |
| **Мемы** | `MemePulseView` | Топ-10 мемов по Heat Score, Volatility Scalper |
| **Сделки** | `TradesView` | Активные / закрытые сделки, Copilot-мониторинг |
| **Радар** | `RadarView` | Полный watchlist, Daily Bias, Fear & Greed, поиск |

### 5.2 Общий поток

```
Сканер (useMexcScanner) ──► signals[] в Zustand
        │
        ├──► Radar — все сигналы
        │
        └──► Sniper — фильтр isSniperQuality()
                    │
                    ▼
            Confidence Score (≥70%, 3/4 факторов)
                    │
                    ▼
            «Открыть сделку» → activeTrades (localStorage)
                    │
                    ▼
            Tactical Copilot (useTradeCopilot) — алерты TP/SL/BE

Meme Pulse (useMemePulseScanner) ──► memeSignals[] (топ-10)
        │
        ▼
    Клик → loadMemeCoinAnalysis() → CoinSignal в signals[]
        │
        ▼
    Tactical Drawer — график, SMC, Meme Pulse панель
```

### 5.3 Tactical Drawer (общий для всех вкладок)

Тап по монете → `selectCoin(symbol)` + `setDrawerOpen(true)` → нижний drawer:

- ProbabilityGauge, RSI, Score, Trend, SL/TP
- LiveChart (+ сессии, зоны, прогноз)
- OrderBookPanel, Whale Watcher, Tape Momentum
- LTF: MSS, Raid, OTE, Absorption, CHoCH, Buyer Aggression
- PO3, Session DNA, BTC Divergence
- Meme Pulse Panel (если сигнал из мем-сканера)
- NewsPanel, Fear & Greed, DataLog

### 6.2 SNIPER MODE — отбор элитных сетапов

**Файлы:** `src/engine/sniperMode.ts`, `src/components/sniper/*`

Отбирает сигналы из общего `signals[]` по критериям:

| Критерий | Условие |
|----------|---------|
| Активный сетап | `hasActiveSetup === true` |
| Score | ≥ 7 / 10 |
| Сила | ≥ 2 из 3: Absorption, LTF CHoCH, Buyer Aggression (LONG) |
| Daily Bias | LONG только при не-BEARISH bias, SHORT — не-BULLISH |
| RSI | LONG: RSI < 45, SHORT: RSI > 55 |
| Risk/Reward | ≥ 1:3 (surgical entry или текущая цена) |

**UI:** `SniperCard` — calibrated win rate, R:R, причины фильтра, кнопки «Открыть сделку» и «Confidence Score».

### 6.3 CONFIDENCE SCORE ENGINE — уверенность входа

**Файл:** `src/engine/confidence/index.ts`

4 фактора → итог 0–100%:

| Фактор | Вес | Содержание |
|--------|-----|------------|
| HTF Context | 30% | Daily Bias + BTC Trend + Coin Trend |
| LTF Structure Flip | 30% | MSS, CHoCH, Raid, OTE |
| Volume Confirmation | 20% | Absorption, Buyer Aggression |
| OrderBook Wall | 20% | BID/ASK стены в нужном направлении |

**Порог входа:** ≥ 70% и ≥ 3/4 факторов → `approved`.

**Качество:** ELITE (>85%), STRONG (>70%), WEAK (<70%).

**UI:** `ConfidenceScore.tsx` — прогресс-бары факторов, рекомендация.

**Инвалидация:** `isTradeInvalidated()` — слом структуры на 1m (Lower Low / Higher High).

### 6.4 TACTICAL COPILOT — сопровождение сделок

**Файлы:** `src/hooks/useTradeCopilot.ts`, `src/components/trades/*`

```
Sniper → Confidence → «Открыть сделку» → Trade Tracker → Copilot
```

**Trade Tracker** (`useAppStore`):
- `activeTrades[]` — localStorage `enterprise_active_trades`, хранение 7 дней
- CRUD: `addTrade`, `updateTrade`, `closeTrade`, `addTradeEvent`

**Copilot проверяет каждые 5 сек:**
| Алерт | Условие |
|-------|---------|
| TP1 / SL | Цена достигла уровня |
| Breakeven | 50% пути до TP1 → «переноси SL в BE» |
| Invalidation | Слом структуры на 1m |
| Wall Watcher | Крупная стена Ask/Bid на пути к TP |

**UI:** `TradesView` (активные / закрытые), `TradeCard` (P&L, прогресс, алерты).

### 6.5 MEME PULSE — Volatility Scalper

**Файлы:** `src/engine/meme/*`, `src/hooks/useMemePulseScanner.ts`, `src/components/meme/*`

```
Все мемы MEXC → фильтр (не blue chip, vol > $5k, price < $15)
        │
        ▼
Глубокий анализ 12 кандидатов → топ-10 в UI
        │
        ▼
3 детектора + Spread Pressure → Heat Score 0–100
```

**Детекторы:**

| Модуль | Файл | Логика |
|--------|------|--------|
| Volume Delta Spike | `volumeSpike.ts` | Объём ×3+ от среднего + движение цены >1.5% |
| Liquidity Gap | `liquidityGap.ts` | «Дырявый» стакан, путь наименьшего сопротивления |
| Mean Reversion | `meanReversion.ts` | RSI >85 / <15 + выход за Bollinger 3σ |
| Spread Pressure | `spreadPressure.ts` | Cumulative delta из ленты сделок (60 сек) |

**Heat Score:** Volume (30) + Gap (25) + Reversion (20) + Pressure (25).

**Рекомендации:** ≥70 CRITICAL/QUICK_ENTRY, ≥50 STRONG, ≥30 MONITOR.

**Клик по мему:** `loadMemeCoinAnalysis()` — полный SMC + meme → `CoinSignal` в store → Tactical Drawer.

**Сигнал:** `buildMemeCoinSignal()` — direction, SL/TP, `hasActiveSetup` при Heat ≥ 50, `activeSignalKey: 'MEME_PULSE'`.

### 6.6 RADAR — обзор рынка

**Файлы:** `src/components/radar/*`

- Сортировка: активные сетапы сверху, затем по `probabilityPct`.
- Daily Bias BTC (1D) + структура BTC 4H.
- Fear & Greed Index.
- `CoinSearch` — поиск и добавление монет в watchlist + one-shot SMC-анализ.
- Sentiment badge на строке монеты.

### 6.7 Probability Engine (SMC)

**Файлы:** `src/engine/smc/index.ts`, `src/engine/ProbabilityEngine.ts`

**Входы:** OHLCV 4H / 1H / 15m / 5m / 1m, daily bias, BTC trend, liquidity map, news boost.

**Выход:** `CoinSignal` — направление, score, %, зоны, уровни, тренд.

**Confluence (база):**

| Фактор | Вес |
|--------|-----|
| Order Block | +3 |
| FVG | +2 |
| Fib OTE | +3 |
| Fib 0.618 / 0.786 | +1 |
| Cap | 10 |

**Цепочка бустов (clamp 0…10):**
`wall → news → liq → divergence → LTF (MSS/Raid/OTE) → absorption → CHoCH`

**LTF-фильтры точности входа:**

| Модуль | Описание |
|--------|----------|
| MSS | Market Structure Shift на LTF |
| Liquidity Raid | Свежий sweep ликвидности |
| OTE Zone | 0.62–0.79 Fib после импульса |
| Absorption (VSA) | Stopping Volume на свече |
| LTF CHoCH 1m | Смена характера + Surgical Entry |
| Buyer Aggression | Buy/Sell ratio из ленты сделок |
| PO3 | Power of Three (Asia Box + фазы дня) |
| Session DNA | Поведение монеты по сессиям |

### 6.8 Live Chart

**Файл:** `src/components/tactical/LiveChart.tsx`

- Таймфреймы: 1m / 5m / 15m / 1h / 4h / 1d.
- Слои: SessionOverlay → свечи LWC → ChartOverlay (OB/FVG) → PredictionOverlay.
- Настройки: индикаторы, зоны, сессии (localStorage).

**Индикаторы:** EMA / SMA / Bollinger / VWAP, RSI, MACD, Stochastic, Volume, ATR.

**Зоны:** Order Blocks, FVG, Fibonacci OTE, POC / Value Area, дневные уровни.

### 6.9 Sessions & Calendar Overlay

- Цветовые полосы: Азия / Лондон / Нью-Йорк / Overlap / выходные (UTC).
- Hardcoded макро: NFP, CPI, FOMC, PCE, EIA.
- На малых ТФ (1m–1h); на 4h/1d скрыто.

### 6.10 Multi-TF Prediction

- Alignment 1D / 4H / 1H.
- Liquidity map + сценарии A/B/C (пути на графике).

### 6.11 Order Book Intelligence

- Глубина стакана, imbalance, история.
- Whale Watcher — крупные стены, события APPEARED/EATEN.
- Tape Momentum — скорость imbalance, burst detection.
- Score boost в Probability Engine.
- 2D heatmap + 3D heatmap (R3F, lazy).

### 6.12 News Intelligence

| Источник | Назначение |
|----------|------------|
| CryptoPanic (public) | Лента крипто-новостей |
| Alternative.me | Fear & Greed |
| RSS CoinDesk / Cointelegraph | Заголовки (через proxy) |

Локальный keyword-sentiment (без LLM) → `scoreBoost` в ProbabilityEngine.

UI: NewsStrip, FearGreedGauge, NewsPanel, SentimentBadge.

## 9. Модели данных

### 9.1 CoinSignal (ядро)

```typescript
interface CoinSignal {
  symbol: string              // PEPEUSDT
  internalSymbol: string      // PEPE/USDT:USDT
  displayName: string         // PEPE/USDT
  price: number
  priceChange24h: number
  currentRSI: number | null
  probabilityPct: number      // 0–100
  score: number               // 0–10
  direction: 'LONG' | 'SHORT' | null
  zones: string[]
  sl, tp1, tp2, tpDaily: number | null
  coinTrend, btcTrend, dailyBias: ...
  hasActiveSetup: boolean
  // LTF расширения
  mss?, raid?, ote?, absorption?, ltfChoCH?, buyerAggression?
  btcDivergence?
  memePulse?: MemeSignal      // если из Meme Pulse
}
```

### 9.2 MemeSignal

```typescript
interface MemeSignal {
  symbol, internalSymbol, displayName, price, priceChange24h
  volumeSpike: VolumeSpikeResult
  liquidityGap: LiquidityGapResult
  meanReversion: MeanReversionResult
  spreadPressure: SpreadPressureResult
  heatScore: number           // 0–100
  quality: 'CRITICAL' | 'STRONG' | 'MODERATE' | 'WEAK'
  recommendation: 'QUICK_ENTRY' | 'MONITOR' | 'WAIT'
}
```

### 9.3 ActiveTrade

```typescript
interface ActiveTrade {
  id, symbol, internalSymbol, direction
  entryPrice, entryTime, sl, tp1, tp2
  status: 'ACTIVE' | 'BREAKEVEN' | 'INVALIDATED' | 'CLOSED_WIN' | ...
  currentPrice, pnlPercent, pnlUsd
  confidenceScore, confidenceFactors
  breakevenAlertShown, invalidationAlertShown, wallAlertShown
  events: TradeEvent[]
}
```

- Моноширинная типографика терминала.
- **Снайпер** — зелёный matrix accent.
- **Мемы** — красный alert accent (огонь, волатильность).
- **Сделки** — matrix + статусные цвета P&L.
- Drawer — плотная телеметрия, минимум декора.

## 12. Риски и дисклеймер

- Источники новостей могут быть недоступны — UI деградирует мягко.
- Keyword-sentiment — эвристика, не NLP-модель.
- Макро-календарь частично hardcoded — даты сверять с официальными релизами.
- Вероятность на экране — **нормализованный confluence**, не статистическая частота побед.
- Meme Pulse — экстремальная волатильность; микро-стопы обязательны.
- Active Trades — локальное сопровождение; исполнение только вручную на бирже.
- Пользователь сам несёт риск на рынке деривативов.

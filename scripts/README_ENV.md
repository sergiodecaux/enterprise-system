# Настройка API ключей MEXC

## Создайте файл `scripts/.env`:

```
MEXC_API_KEY=your_api_key_here
MEXC_SECRET_KEY=your_secret_key_here
```

## Получение API ключей:

1. Зайдите на https://www.mexc.com/user/openapi
2. Создайте API ключ
3. Скопируйте API Key и Secret Key в файл `.env`

## Пример `.env`:

```
MEXC_API_KEY=mx0vglVwI7AVA3rBEE
MEXC_SECRET_KEY=65490319b47b45edb9ef420320497475
```

**ВАЖНО**: Файл `.env` уже добавлен в `.gitignore` и не будет загружен в репозиторий.
